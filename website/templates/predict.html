How to Use the Model

  1. Download one of the sample image zip files.
  2. Extract the zip file to get the images.
  3. Upload an image on the "Model" page using the "Upload Image" button.
  4. Wait for the model to process the image. This may take a few moments.
  5. Once the image has been processed, the results will be displayed on the screen.
  6. You can view the original image, the importance image, and the model's prediction.
  7. If the important region is incorrect, simply select a color and draw on top of the image to show the correct region
  8. If you want to try another image, simply repeat the process.
